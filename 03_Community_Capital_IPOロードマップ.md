# Community Capital IPO ロードマップ
## ── ファンダムと影響力を資本に変え、全員で「本のエコシステム」のオーナーになる ──

---

## コンセプト

> **これまでのIPOは「投資家のもの」だった。**
> **バリューブックスのIPOは「本を愛する人たちのもの」にする。**

VCが入っていないため、上場時の株価を無理に吊り上げる必要がない。
「資金調達のためのIPO（Exit）」ではなく、**「最強のコミュニティを作るためのIPO（Co-ownership）」**。

---

## Phase 1：上場前夜 ── 最強の「身内」を増やす（20〜23周年）

### 1-1. コンセプト

この3年間で、売上を40億から100億超へ持っていくための**「エンジン」**として株式を使う。

大手に大きなお金を入れてもらうのではなく、**影響力を持った個人、またはインディペンデントな出版社・書店に集客をしてもらう。** お金と倉庫やシステムなどはバリューブックスが用意して、共に一気に成長できる仲間を増やしていく。

### 1-2. 外部パートナーへの巻き込み

#### 対象パートナー候補

| カテゴリ | 具体的な候補 | 巻き込み方 |
|---------|------------|-----------|
| **クリエイター・インフルエンサー** | ゆる言語学ラジオ、古典ラジオ、三宅夏帆、ジェーン・スー | 有償新株予約権 |
| **書店** | 本屋B&B（内沼晋太郎）、他の独立系書店 | 株式交換・第三者割当 |
| **出版社** | ミシマ社、エイジ出版 | 業務資本提携 |
| **書店オーナー・プロデューサー** | 内沼晋太郎 | 株式交換＋顧問的関与 |

#### メッセージ

> **「広告費としてお金を払うのではなく、将来のバリューブックスの価値（株）をお渡ししたい。一緒に時価総額400億を目指しませんか？」**

これにより、彼らがSNSで発信するモチベーションが**「案件」から「自分たちの会社の宣伝」**に変わる。

### 1-3. 社内キーマンへの資産形成

| 対象 | 手法 | 目的 |
|-----|------|------|
| **飯田光平氏ら経営幹部** | ストックオプション（創業メンバー級の付与量） | 100億〜200億への牽引力 |
| **コミット社員（継続的に関わりたい人）** | 従業員持株会＋SO | 長期定着、上場益の共有 |
| **清水氏（共同創業者）** | 既存2%の維持＋追加SO検討 | 創業への敬意 |

> 「24周年に上場する」という明確なゴール（とリターン）があることで、100億〜200億への急成長に伴う強烈な負荷（「産みの苦しみ」）を乗り越える求心力になる。

### 1-4. B&Bとの業務資本提携（Phase 1の最重要マイルストーン）

| 項目 | 内容 |
|-----|------|

## Phase 2：IPO本番 ── ファンが株主になる日（24周年＝2031年頃）

### 2-1. 上場市場の選択

| 市場 | メリット | デメリット |
|------|---------|----------|
| **東証グロース** | 上場基準が緩い、成長企業向け | 流動性が低い |
| **東証プライム** | 信用力・流動性が高い | 基準が厳しい |
| **推奨：グロース→プライム** | まずグロースで上場し、300億規模でプライムへ | 段階的に信用を獲得 |

### 2-2.「親引け」の最大活用

#### 親引けとは

通常、IPOの株は証券会社が選んだ投資家に配られるが、**「親引け」**という制度を使えば、会社が指定した相手に優先的に株を売ることができる。

#### バリューブックスの親引け戦略

| 割当先 | 想定割当比率 | 狙い |
|-------|------------|------|
| **提携書店連合**（B&B等） | 3〜5% | 安定株主の確保 |
| **提携出版社** | 2〜3% | サプライチェーンの固定化 |
| **クリエイター・著者** | 2〜3% | 「自分の会社」意識の強化 |
| **ロイヤルユーザー**（年間買取数・購入数の多い顧客） | 1〜2% | ファンの株主化 |
| **一般公募** | 残り | 市場からの資金調達 |

> **「日本初、VCではなくファンが主要株主のIPO」** として、メディアの注目を集めるブランディングにもなる。

#### 実務上の注意点（→ 詳細は [04_法的論点](./04_法的論点と監査法人レビュー.md)）

- 親引け先は証券会社と東証の承認が必要
- 個人への大量の親引けは証券会社が事務負担を嫌うため、**BtoBパートナー（書店・出版社）に絞る**のが現実的
- ファン個人には通常の「IPO申し込み」をしてもらい、**株主優待で優遇**する方が実務的

### 2-3. 時価総額400億の根拠

| 項目 | 数値 | 根拠 |
|-----|------|------|
| 売上高 | 200億円 | Phase 1での成長実績 |
| 営業利益 | 20億円 | 営業利益率10% |
| PER | 20倍 | 流通小売・テック企業として妥当 |
| **想定時価総額** | **400億円** | 20億 × 20 = 400億 |
| プレミアムケース | 500〜600億円 | PER 25〜30倍（コミュニティの「堀」を評価） |

### 2-4. Phase 2のKPI

| 指標 | 目標 |
|-----|------|
| 上場市場 | 東証グロース |
| 公募・売出し総額 | 50〜100億円 |
| 親引け比率 | 10〜15%（パートナー株主向け） |
| 上場時時価総額 | 400億円 |
| 創業者持株比率 | 40〜50% |
| パートナー株主比率 | 10〜15% |

## Phase 3：上場後 ── 貢献する人が報われ続ける仕組み（25周年〜）

### 3-1. 貢献連動型株主優待

通常の「株主全員に同じモノを送る」優待ではなく、**事業への貢献度に応じてランクが変わる**独自の優待制度。

#### 基本優待（全株主）

| 優待内容 | 詳細 |
|---------|------|
| 買取査定額アップ | +10% |
| 購入時割引 | 5%OFF |
| 株主限定ニュースレター | 月次の事業レポート |

#### ゴールド優待（貢献度の高い株主）

| 条件 | 追加優待 |
|-----|---------|
| 年間買取数100冊以上 | 査定額 +20%、限定イベント招待 |
| プロモーターとして年間100万円以上の売上貢献 | 希少本の優先購入権、株主総会VIP席 |
| パートナー書店・出版社 | **VB物流の株主特別レート**（通常の20%OFF） |

> **株主優待が「バリューブックス経済圏のパスポート」になる。**

### 3-2. 株主総会を「本のフェス」にする

| 要素 | 内容 |
|-----|------|
| **場所** | 安曇野 or 上田の倉庫/NABO |
| **形式** | 総会＋トークイベント＋本の市＋著者サイン会 |
| **参考事例** | スノーピーク「スノーピークウェイ」 |
| **狙い** | 株主を「投資家」ではなく「ファミリー」に |

### 3-3. 継続的な「第三者割当」による仲間づくり

上場企業になれば、いつでも市場価格で株を発行して資金調達やM&Aができる。

| 場面 | 手法 | 例 |
|------|------|-----|
| **新パートナーの発掘** | 第三者割当増資 | 次世代のインフルエンサー、新しい独立系書店 |
| **書店のグループ化** | 株式交換によるM&A | 個性的な書店を「バリューブックス連邦」に迎え入れ |
| **出版社との深化** | 資本業務提携 | サプライチェーンの固定化 |

> **これを繰り返すことで、バリューブックスは単なる古書店ではなく、「出版文化を支えるインフラ企業グループ（ホールディングス）」へと進化する。**

### 3-4. 「バリューブックス・ホールディングス」の将来像

```
┌────────────────────────────────────────────────┐
│          バリューブックス・ホールディングス          │
│          （東証プライム・時価総額600億〜）         │
├────────────────────────────────────────────────┤
│                                                │
│  ┌──────────┐ ┌──────────┐ ┌──────────────┐   │
│  │ VB本体   │ │ パートナー │ │ 出版・メディア │   │
│  │ (物流/EC) │ │ 書店連合  │ │ 事業         │   │
│  └──────────┘ └──────────┘ └──────────────┘   │
│                                                │
│  ┌──────────┐ ┌──────────┐ ┌──────────────┐   │
│  │ 財団     │ │ VBラボ   │ │ 海外事業     │   │
│  │ (社会貢献)│ │ (R&D)   │ │ (将来構想)    │   │
│  └──────────┘ └──────────┘ └──────────────┘   │
│                                                │
│  株主構成：                                     │
│  創業者 30-35% ／ パートナー連合 35-40%           │
│  ／ 市場 25-35%                                │

```

## 20周年での宣言案

> **「私たちはこれまで、独立性を守るために上場を避けてきました。**
> **しかし、100億、300億という規模で『本のエコシステム』を作るには、私ひとりの手では足りません。**
>
> **24周年、バリューブックスは上場します。**
> **ただし、知らない誰かにお金を出してもらうためではありません。**
> **著者、出版社、書店、そして本を愛する皆さんに、この会社のオーナーになってもらうためです。**
>
> **倉庫とシステムと資金は、私たちが用意しました。**
> **どうか、皆さんの『影響力』と『偏愛』を貸してください。**
> **一緒に、本の世界を変える会社を共有しましょう。」**
