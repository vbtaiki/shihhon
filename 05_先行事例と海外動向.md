# 先行事例と海外動向
## ── 「ステークホルダー所有型企業」の世界的潮流とバリューブックスへの示唆 ──

---

## 1. エグゼクティブ・サマリー

バリューブックスの「Community Capital IPO」構想は、前例のない挑戦に見えるが、実は世界的な潮流と合致している。以下の3つの流れが、この構想の正当性を裏付ける：

1. **ステークホルダー・キャピタリズムの台頭**（2019年〜）
2. **上場時のステークホルダー優先割当の実例**（Airbnb、Duolingo等）
3. **日本における「ほぼ日」型のコミュニティIPO**

---

## 2. 国内事例

### 2-1. 株式会社ほぼ日 ── 最も近い成功例

| 項目 | 内容 |
|-----|------|
| **モデル** | 「場」の永続性のための上場 |
| **上場年** | 2017年（JASDAQ、現東証スタンダード） |
| **特徴** | VCを入れず、自己資本で成長してから上場 |
| **株主構成** | 糸井重里氏が大株主を維持しつつ、ファン・顧客が広く保有 |
| **株主優待** | ほぼ日手帳など、ファンが喜ぶ実用品 |
| **株主総会** | ファンとの対話の場として機能 |

#### バリューブックスへの示唆

| ほぼ日の選択 | VBへの応用 |
|-------------|-----------|
| 個人の会社から「公器」への転換 | 中村個人の会社から「出版エコシステムの公器」への転換 |
| VCなしで上場 | **バリューブックスと同じ境遇** |
| ファンとの丁寧な対話 | 株主総会＝「本のフェス」化 |
| 糸井氏の持株比率維持 | 中村氏が34%以上を維持し、理念を守る |

> **ほぼ日は「VCなし上場」の日本における最良の前例。**
> バリューブックスは、ここに「パートナーへの株式付与」という要素を加える。

---

### 2-2. 株式会社スノーピーク ── ユーザー＝株主のコミュニティ化

| 項目 | 内容 |
|-----|------|
| **モデル** | 「ユーザー＝株主」のコミュニティ化 |
| **上場年** | 2014年（東証マザーズ、のちプライム） |
| **特徴** | 「スノーピーク・ウェイ」というキャンプイベントで、ユーザーと直接対話 |
| **株主層** | 熱狂的なユーザー（スノーピーカー）が株主に。長期保有率が高い |
| **その後** | 2024年にMBOで非上場化（株主構成の再設計が動機の一つ） |

#### バリューブックスへの示唆

| スノーピークの学び | VBへの応用 |
|------------------|-----------|
| 株主総会をファンイベントに | 安曇野/上田での「本のフェス」株主総会 |
| ユーザー株主が安定株主として機能 | 買取・販売のロイヤルユーザーを株主に |
| MBOの教訓：機関投資家との軋轢 | **上場時からパートナー株主比率を高く設計し、機関投資家主導にならないようにする** |

---

### 2-3. CYBERDYNE ── 種類株式上場の唯一の前例

| 項目 | 内容 |
|-----|------|
| **モデル** | 黄金株（拒否権付種類株式）での上場 |
| **上場年** | 2014年（東証マザーズ） |
| **特徴** | 創業者（山海教授）が議決権の多い種類株式を保有し、経営の独立性を維持 |
| **東証が認めた理由** | 「革新的な医療技術の流出防止」という極めて高い公益性 |
| **条件** | サンセット条項（一定期間後に普通株に転換される仕組み） |

#### バリューブックスへの示唆

- 「出版文化の保護」がCYBERDYNEの「技術流出防止」と同等の公益性として認められるかは**未知数**
- **推奨は「黄金株ではなく普通株での安定保有」**だが、「ルールに挑戦する」選択肢もゼロではない

---

### 2-4. クックパッド ── パートナー株主の失敗例としての教訓

| 項目 | 内容 |
|-----|------|
| **教訓** | 創業者が持株比率を下げすぎた結果、経営権を奪われた |
| **何が起きたか** | 創業者（佐野氏）の持株比率低下後、外部株主による経営介入が発生 |

#### バリューブックスへの示唆

> **創業者は最低34%（拒否権ライン）を絶対に下回ってはいけない。**
> 「パートナーに株を渡す」美しい構想と、「経営権を守る」現実的な防衛は両立させる必要がある。

## 3. 海外事例

### 3-1. Airbnb ── ホストへのIPO株優先割当

| 項目 | 内容 |
|-----|------|
| **モデル** | ステークホルダーへの優先割当（Directed Share Program） |
| **上場年** | 2020年（NASDAQ） |
| **施策** | 上場時、株式の約7%（約2.3億ドル相当）を、ビジネスを支えてきた**「ホスト（家主）」**のために確保 |
| **仕組み** | ホストが公募価格でIPO株を購入できる枠を設定 |
| **結果** | ホストが「自分たちの会社」として株を保有し、ロイヤリティが向上 |

#### バリューブックスへの示唆

| Airbnbの施策 | VBへの応用 |
|-------------|-----------|
| ホスト＝サービスの供給者に株を優先割当 | **プロアカウントのパートナー（サプライヤー・プロモーター・バイヤー）に株を優先割当** |
| 公募価格で購入可能 | パートナーにIPO株を公募価格で購入する権利を付与 |
| ステークホルダー＝株主 | 「本を売る人」「本を紹介する人」「本を仕入れる人」が株主に |

> **Airbnbのモデルは、Community Capital IPOの最も直接的なグローバル先行事例。**

---

### 3-2. Duolingo ── ユーザー・ボランティアへのIPO株

| 項目 | 内容 |
|-----|------|
| **モデル** | ユーザーと貢献者への優先割当 |
| **上場年** | 2021年（NASDAQ） |
| **施策** | 語学学習アプリのユーザーや、コース作成に貢献したボランティアに、IPO株を購入できる優先権を付与 |

#### バリューブックスへの示唆

| Duolingoの施策 | VBへの応用 |
|---------------|-----------|
| ボランティア貢献者に株式 | チャリボン・ブックギフトの長期支援者にIPO株の購入権 |
| ユーザーコミュニティの「自分ごと」化 | バリューブックスの買取・購入ヘビーユーザーを株主に |

---

### 3-3. パタゴニア ── スチュワード・オーナーシップの極致

| 項目 | 内容 |
|-----|------|
| **モデル** | 全株式を環境保護団体に譲渡（2022年） |
| **創業者** | イヴォン・シュイナード |
| **施策** | 議決権株式を「パタゴニア・パーパス・トラスト」に、残りを「ホールドファスト・コレクティブ」に譲渡 |
| **結果** | 年間利益（約1億ドル）が全額、地球環境保護に充てられる |

#### バリューブックスへの示唆

- パタゴニアは**非上場のまま**この転換を行った
- バリューブックスが**上場を選ぶ場合**、パタゴニア型の完全譲渡は難しいが、「財団による安定保有」で精神は受け継げる
- イヴォン・シュイナードの思想は中村氏のOSに既にインストール済み（→ [経営哲学と次世代継承](../../03_リサーチ・コンテンツ系/読書体験最大化エコシステム構想/03_経営哲学と次世代継承.md)）

---

### 3-4. John Lewis Partnership（英国） ── 従業員所有企業

| 項目 | 内容 |
|-----|------|
| **モデル** | 全従業員が「パートナー」として会社を所有 |
| **規模** | 英国最大の百貨店チェーン（売上約1.5兆円） |
| **仕組み** | 利益を従業員にボーナスとして分配（年間給与の8〜20%相当） |
| **ガバナンス** | 従業員議会が経営に関与 |

#### バリューブックスへの示唆

| John Lewisの学び | VBへの応用 |
|-----------------|-----------|
| 従業員＝オーナーという文化 | スタッフ全員が「パートナー」として会社を所有する文化 |
| 利益分配のメカニズム | 持株会＋利益連動型ボーナス |
| 長期的な安定経営 | 「四半期ごとの株価」に振り回されない経営 |

---

### 3-5. Bookshop.org（米国） ── 独立系書店のためのプラットフォーム

| 項目 | 内容 |
|-----|------|
| **モデル** | 独立系書店がオンラインで売上を得るプラットフォーム |
| **設立** | 2020年 |
| **仕組み** | 顧客が「応援したい書店」を選んで本を購入、売上の一部がその書店に |
| **実績** | 3,000以上の書店が参加、累計35億円以上を独立系書店に還元 |
| **組織形態** | **B Corp認証取得済み**（バリューブックスと同じ） |

#### バリューブックスへの示唆

| Bookshop.orgの特徴 | VBへの応用 |
|-------------------|-----------|
| 独立系書店のEC支援 | プロアカウントのバイヤー機能と同コンセプト |
| B Corp × 書店支援 | バリューブックスも同じ立ち位置 |
| **課題：資本関係がない** | **VBは「株式」で書店との資本関係を構築**し、Bookshop.orgより深い連帯を作れる |

> **Bookshop.orgが「業務提携」で止まっている独立系書店支援を、バリューブックスは「資本提携」にまで昇華させる。**

## 4. 思想的背景：ステークホルダー・キャピタリズム

### 4-1. ビジネス・ラウンドテーブル声明（2019年）

| 項目 | 内容 |
|-----|------|
| **何が起きたか** | 米国の主要181社のCEOが連名で「株主至上主義の終焉」を宣言 |
| **主張** | 企業は株主だけでなく、顧客、従業員、サプライヤー、地域社会のすべてのステークホルダーのために存在する |
| **署名者** | Apple、Amazon、JPMorgan等の米国最大手企業CEO |

### 4-2. ESG投資の拡大

| 指標 | 数値 |
|-----|------|
| 世界のESG運用資産 | 約40兆ドル（2024年） |
| 日本のESG投資 | 約500兆円規模（GPIF等） |

> **バリューブックスの「パートナーを株主にする」構想は、ESG投資家にとって極めて魅力的なストーリー。**
> B Corp認証＋ステークホルダーオーナーシップは、世界的にも先端的な企業モデル。

### 4-3. Purpose-Driven Company（目的主導企業）の台頭

| 企業 | 目的（パーパス） | 株主構成の特徴 |
|------|----------------|---------------|
| パタゴニア | 地球を救うためにビジネスを営む | 全株を財団に譲渡 |
| ほぼ日 | 人がうれしくなるものを作る | 創業者＋ファン株主 |
| バリューブックス（構想） | **本の循環を通じて、世界を豊かにする** | **創業者＋パートナー＋ファン株主** |

## 5. 事例横断分析：バリューブックスのポジショニング

### 「所有の分散度」×「上場/非上場」マトリクス

```
                    非上場              上場
                 ┌────────────────┬────────────────┐
  株主が         │                │                │
  少数（集中）   │  現在のVB      │  通常の上場企業  │
                 │  (創業者98%)   │                │
                 ├────────────────┼────────────────┤
  株主が         │  パタゴニア    │  ★ VBの目標    │
  多数（分散）   │  John Lewis   │  （ほぼ日の     │
                 │               │   進化形）      │
                 └────────────────┴────────────────┘
```

> **バリューブックスが目指すのは、右下の象限。**
> 上場企業でありながら、株主が「投資家」ではなく「パートナー」で構成される、新しい企業モデル。

## 6. 「出版業界」特有の資本提携事例

### 日本国内

| 事例 | 内容 | 示唆 |
|------|------|------|
| **ブックセラーズ＆カンパニー** | 紀伊國屋・CCC・日販の合弁（2023年） | 大手同士の提携。VBは「独立系」としての対抗軸 |
| **メディアドゥ** | 電子書籍取次として上場 | 「出版流通インフラ」として上場した先例 |
| **トーハン × 日販** | 物流共同化の検討 | 既存取次の限界が、VBの「第三の道」を正当化 |

## 7. まとめ：バリューブックスの構想の正当性

### 7-1. 世界的潮流との整合性

| 潮流 | VBの構想 |
|------|---------|
| ステークホルダー・キャピタリズム | パートナーを株主にする |
| ESG投資の拡大 | B Corp × ステークホルダーオーナーシップ |
| Purpose-Driven Company | 出版文化の保護が企業の目的 |
| Directed Share Program（IPO優先割当） | 親引けによるパートナーの株主化 |

### 7-2. 先行事例から学ぶべきこと

| 学び | 出典 |
|-----|------|
| VCなしでも上場できる | ほぼ日 |
| ステークホルダーにIPO株を優先割当できる | Airbnb, Duolingo |
| ユーザー＝株主のコミュニティは作れる | スノーピーク |
| 創業者の持株比率は絶対に守る | クックパッド（反面教師） |
| 出版インフラ企業は上場できる | メディアドゥ |
| B Corp × 書店支援は成立する | Bookshop.org |

### 7-3. バリューブックスだけの独自性

| 独自性 | 内容 |
|-------|------|
| **VCゼロで40億円企業** | 株主構成を100%自分で設計できる |
| **物流インフラ（SKYPOD倉庫）** | テクノロジーの「堀」がある |
| **B Corp認証** | 社会的企業としての国際的信用 |
| **メディア（積読チャンネル等）** | 自社でファンコミュニティを持つ |
| **プロアカウント構想** | パートナーを巻き込む「仕組み」がすでにある |
| **出版業界のDX** | 経産省が推進する書店振興との連動が可能 |

> **これらの要素をすべて持つ企業は、日本に他にない。**
> **だからこそ、「Community Capital IPO」はバリューブックスにしかできない挑戦であり、成功すれば日本の資本市場の歴史に名を刻む事例となる。**
