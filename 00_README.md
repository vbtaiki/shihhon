# 資本政策 ── ステークホルダーオーナーシップ構想

## このフォルダについて

バリューブックスの**20周年以降の資本政策**と、**「ステークホルダー所有型企業」への転換構想**をまとめた資料群です。

プロアカウントを通じた著者・書店・出版社・インフルエンサーの巻き込みから、株式交換・ストックオプションによるパートナーの株主化、そしてIPOを手段として活用した「Community Capital IPO」構想まで、バリューブックスの次の10年を設計するための戦略資料です。
